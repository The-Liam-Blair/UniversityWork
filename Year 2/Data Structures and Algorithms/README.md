Coursework is ongoing. Will be written in C and intended to be compiled and run on the Microsoft Compiler.

Coursework is to design a connect 4 game, while utilising multiple sensible data structures to maximise efficiency.
Also may include optoinal features, such as an undo / redo button, a computer AI for the 2nd player, resizing the board,
and recording previous games and allowing the user to automatically replay a past saved game.
